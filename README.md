# template-revtex

## What's this?

LaTeX プロジェクトのためのテンプレート。
- インストール自動化 → nix
- 自動化作業(build/clean) → [Makefile](./Makefile)

## Usage
### Install

1. `git clone <this repo>`
1. `make install`

### Build 

1. `make`
